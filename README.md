# PyBank and PyPoll Analysis

This repository contains Python scripts for analyzing budget and election data, PyBank and PyPoll respectively.

## PyBank Analysis

The PyBank script analyzes financial data to provide insights such as total months, total profits/losses, average change, and the greatest increase and decrease in profits.

#### How to Run:
- Ensure the `budget_data.csv` file is in the specified path.
- Execute the script `main.py`
- Results will be printed to the terminal and exported to `budget_analysis.csv`.

#### Results:
- Total Months: 86 Months
- Total Profits/Losses: $22564198
- Average Change: $-8311.11
- Greatest Increase in Profits: Aug-16 ($1862002)
- Greatest Decrease in Profits: Feb-14 ($1825558)

## PyPoll Analysis

The PyPoll script analyzes election data to provide insights such as total votes cast, each candidate's percentage of votes, and the winner of the election.


#### How to Run:
- Ensure the `election_data.csv` file is in the specified path.
- Execute the script `main.py`.
- Results will be printed to the terminal and exported to `election_analysis.csv`.

#### Results:
- Total Votes: 369711
- Charles Casper Stockham: 23.049% (85213)
- Diana DeGette: 73.812% (272892)
- Raymon Anthony Doane: 3.139% (11606)
- Winner:Diana DeGette

## Accessing Results

Once the scripts are executed, the results will be exported to the specified CSV files in the respective `analysis` folders. These files will show a charted view of the results to review. You can access these files using the provided local paths.
